📚 Algorithm Explanation

Hadamard Test: A quantum algorithm to estimate the real part of ⟨ψ|U|ψ⟩ where U is a unitary operator.

Steps:

  1. Prepare ancilla qubit in |+⟩ = (|0⟩ + |1⟩)/√2 using Hadamard gate
  2. Apply controlled-U operation with ancilla as control
  3. Apply Hadamard to ancilla again
  4. Measure ancilla: P(|0⟩) = (1 + Re(⟨ψ|U|ψ⟩))/2

Phase Kickback: When the controlled gate acts on an eigenstate, the phase is "kicked back" to the control qubit, enabling phase estimation.
