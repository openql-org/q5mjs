# Q5M.js + p5.js Integration

Interactive quantum circuit visualizer built with p5.js creative coding library.

## Features

- **Interactive Circuit Building**: Click to add quantum gates
- **Real-time Visualization**: See quantum states update instantly
- **Animated Background**: Beautiful particle effects
- **State Analysis**: Complex amplitude plotting and probability visualization
- **Preset Circuits**: Quick access to Bell states, GHZ states, and superposition

## Files

- `quantum-visualizer.html` - Complete interactive visualizer
- `README.md` - This documentation

## Usage

1. Open `quantum-visualizer.html` in a web browser
2. Use the control buttons to add gates to the quantum circuit
3. Watch the real-time visualization of quantum states
4. Try preset circuits for common quantum phenomena

## Controls

- **Gates**: Add Hadamard and Pauli-X gates to individual qubits
- **Entanglement**: Create CNOT gates between qubit pairs
- **Control**: Reset circuit, pause/play animation, export circuit
- **Presets**: Load common quantum states (Bell, GHZ, Superposition)

## Visualizations

1. **Probability Circles**: Circular representation of basis state probabilities
2. **State Vector**: Complex plane visualization of quantum amplitudes  
3. **Probability Bars**: Bar chart of measurement probabilities
4. **Circuit Diagram**: Visual representation of the quantum circuit

## Technical Details

- Uses p5.js for canvas-based rendering and animation
- Integrates Q5M.js quantum computing library via CDN
- Responsive design works on desktop and mobile
- Real-time performance optimized for smooth animation
