q5m.js Examples

Quantum Computing Simulations in TypeScript

Quick Start

import { Circuit } from '@q5m/q5m';

// Create a 2-qubit circuit
const circuit = new Circuit(2);

// Create Bell state
circuit
  .h(0)           // Hadamard on qubit 0
  .cnot(0, 1);    // CNOT with control=0, target=1

// Execute and get results
const result = circuit.execute();
console.log('Bell state created:', result.state.toString());

Getting Started

# Install q5m.js
npm install q5m

# Run examples
npm run examples

# Start development server
npm run dev

# Build for production
npm run build
