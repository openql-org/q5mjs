📋 API Reference

Available Q5M.js Methods

  • Circuit Creation: new Q5M.Circuit(numQubits)
  • Single-qubit gates: h(), x(), y(), z(), s(), t(), sdg(), tdg(), rx(), ry(), rz(), phase()
  • Two-qubit gates: cnot(), cx(), cz(), cy(), ch(), swap(), cp()
  • Measurement: mz(), mx(), my()
  • Execution: execute()
  • State methods: probabilities(), amplitudes()
